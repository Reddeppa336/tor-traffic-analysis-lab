# Tor Traffic Analisis lab

\###Tor Traffic Investigation and Detection Lab###


Overview:


This project demonstrates how to investigate and detect Tor network usage from a Linux endpoint using host-based and network-based forensic techniques.


Tools Used:


-Tor Browser
-Wireshark
-tcpdump
-ss
-journalctl
-Investigation Workflow
-Process identification
-Port enumeration
-Connection analysis
-Log review
-Packet capture
-IOC identification
-Key Findings
-Tor process identified
-SOCKS proxy detected on port 9050
-Multiple encrypted relay connections observed
-Circuit establishment confirmed through logs
-Skills Demonstrated
-Network Traffic Analysis
-Linux Forensics
-Threat Hunting
-SOC Investigation
-Tor Detection Techniques
